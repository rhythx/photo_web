# Photography Portfolio Website

A minimalist, high-performance photography portfolio featuring:
- **"High Cold" Aesthetic**: Deep dark theme with refined typography.
- **Professional Lightbox**: Supports EXIF data display.
- **Admin Panel**: Drag & drop uploads, photo management, and Series organization.
- **Backend**: Node.js + Express + Sharp (Image Optimization).

## Setup
1. `npm install`
2. `node server.js`
3. Visit `http://localhost:3000`
